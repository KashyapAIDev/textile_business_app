# 🧵 Vastra - Textile Business App

A modern Flutter application designed for textile businesses to showcase products, categories, and allow customers to send inquiries directly through the app.

---

## 📱 About

Vastra is a simple and user-friendly textile business application built with Flutter. It helps textile manufacturers, wholesalers, and retailers display their products digitally while making it easy for customers to connect through WhatsApp and inquiry forms.

---

## ✨ Features

- 🏠 Beautiful Home Screen
- 🧵 Product Categories
- 📦 Product Listings
- 🔍 Product Details
- 📋 Customer Inquiry Form
- 💬 Direct WhatsApp Contact
- 📱 Responsive UI
- ⚡ Fast Performance
- 🎨 Clean Material Design

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

### Installation

Clone the repository

```bash
git clone https://github.com/yourusername/vastra-textile-business.git
```

Go to project directory

```bash
cd vastra-textile-business
```

Install packages

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
